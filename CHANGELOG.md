# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-17

### Added
- Initial release of the ThreadSync Simulator.
- Support for Many-to-One, One-to-One, and Many-to-Many threading models.
- Interactive Mutex Lock and Synchronization demos.
- High-fidelity visualizer with real-time CPU utilization metrics.
- JSDoc documentation for core simulation logic.
- MIT License and project metadata.

### Improved
- README documentation with detailed architecture overview.
- Prettier configuration for better code consistency.
